表格 table 
        1.表格作用，用来显示数据
        2.table相当于一个表格
        3.table标签中使用tr声明表格包含多少行，一个tr代表一行
        4.在tr中声明td,指定每一行有多少个单元格（有多少列）

table属性 border:设置表格边框 width/height:设置表格宽高 align:设置表格对齐方式：left/right/center cellpadding:设置单元格边框和文本内容间的间距 callspacing:设置单元格和单元格之间的间距 tr: align:设置每一行的文本水平排列方式 left/right/center valign:设置每一行中的文本垂直排列方式 td:在tr中声明td
